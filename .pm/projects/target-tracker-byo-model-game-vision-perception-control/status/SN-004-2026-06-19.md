## North star
A trustworthy bring-your-own-model game-vision training service: a user captures their own single-player/offline game, the funnel turns it into a versioned, provenance-clean dataset, and they get back a specialist ONNX whose accuracy is a **measured, reproducible** number (scene-disjoint held-out mAP backed by an in-game hit-rate floor) — all enforced offline-only with the anti-cheat/hardware-passthrough guardrail structural, not just textual. Success = ≥1 game (ideally a non-humanoid second) taken end-to-end to a model that passes a written, owner-ratified reliability bar, on a general base model that demonstrably reduces the per-game data a user must supply.
